# Parallel Programming Kit

# Installation for Python

## Developer mode

1. clone repository
2. cd into it
3. install it as local editable package: `python3 -m pip install -e .`

```
git clone https://github.com:pavelvasev/ppk.git
# if have github account: git clone git@github.com:pavelvasev/ppk.git
cd ppk
python3 -m pip install -e .
```

## User mode

PPK is available as python package via pip.
```
pip3 install ppk
```
